**日期**：2026-02-25
**范围**：product-audit-skill — product-map 技能
**目标**：在现有 Markdown 报告之外，新增两个独立 SVG 文件，让人类更直观理解功能地图和业务流

## 一、product-map-visual.svg — 角色-任务树

### 布局

水平树形布局：左列 = 角色，右侧 = 该角色的任务列表。

```
[角色1] ─┬─ [任务A]
         ├─ [任务B]
         └─ [任务C]

[角色2] ─┬─ [任务D]
         └─ [任务E]
```

- 角色列宽：160px，任务列起点：220px
- 任务框宽：220px，高：36px，间距：10px
- 每个角色组垂直间距：24px
- 画布宽：固定 480px，高：按内容动态计算

### 视觉规格

| 元素 | 颜色 | 尺寸 | 说明 |
|------|------|------|------|
| 角色框 | `#3B82F6`（蓝）| 150×40px | 白色字体，居中，圆角 6px |
| 任务框·高频 | `#22C55E`（绿）| 220×36px | 白色字体，圆角 4px |
| 任务框·中频 | `#F59E0B`（黄）| 220×36px | 白色字体，圆角 4px |
| 任务框·低频 | `#9CA3AF`（灰）| 220×36px | 白色字体，圆角 4px |
| 连线 | `#CBD5E1` | 1px | 角色框右中 → 任务框左中的折线 |
| 风险徽章 | `#EF4444`（红）| 8px 圆 | 高风险任务右上角 |
| 跨部门徽章 | `#8B5CF6`（紫）| 8px 圆 | cross_dept=true 时显示，紧贴风险徽章左侧 |
| 冲突 Flag | `#F97316`（橙）| 10px 三角 | CONFLICT/CRUD_INCOMPLETE flags 不为空时，右上角三角 |

### 图例

SVG 底部附图例，说明颜色含义（高频/中频/低频/高风险/跨部门/冲突）。

### 坐标计算公式

```
角色框 y = 顶部 padding(40) + 累计前面所有角色组高度
角色组高度 = max(40, 任务数 × (36 + 10) - 10) + 24
任务框 y = 角色框中心 y - (任务数-1) × (36+10)/2 + i × (36+10)
```

## 二、business-flows-visual.svg — 业务流泳道图

### 布局

每条业务流一个泳道区块，多条流竖向叠放，流之间用分割线和标题分隔。

每条流内部：
- 左侧 = 泳道标签列（角色/系统名），宽 100px
- 右侧 = 节点区，水平排列

```
流名称: F001 售后全链路
─────────────────────────────────────────────
买家        [节点1] ──────────────→ [节点4⚠]
商户后台             [节点2] → [节点3]
─────────────────────────────────────────────
流名称: F002 订单支付
...
```

### 视觉规格

| 元素 | 颜色/样式 | 说明 |
|------|----------|------|
| 流标题栏 | `#1E293B`（深灰）背景，白字 | 显示 flow id + name |
| 泳道标签 | `#F1F5F9` 背景，`#475569` 文字 | 100px 宽 |
| 泳道分割线 | `#E2E8F0`，1px 虚线 | 每条泳道之间 |
| 正常节点框 | `#3B82F6` 边框，`#EFF6FF` 填充 | 150×40px，圆角 6px |
| 缺口节点框 | `#EF4444` 虚线边框，`#FEF2F2` 填充 | 同尺寸，右上角 ⚠ |
| handoff 箭头 | `#64748B`，实线 + 箭头 | 节点间连接 |
| handoff 标签 | `#64748B`，10px 字体 | 箭头上方显示机制（webhook/event 等） |
| seq 序号 | 节点左上角白底蓝字小圆 | 显示 1/2/3... |

### 节点坐标计算

```
每条泳道高度：60px（节点 40px + 上下 padding 各 10px）
流区块高度：流标题 32px + 参与泳道数 × 60px + 底部 16px
节点 x = 100(泳道标签) + 20(左边距) + (seq-1) × (150+40)
节点 y = 流起点 y + 32(标题) + 泳道index × 60 + 10
```
