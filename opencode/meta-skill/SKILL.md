# Meta-Skill v0.2.0

> Unified skill generator — /bootstrap + /run.

## Commands

| Command | Purpose |
|---------|---------|
| bootstrap | Analyze project → generate node-specs + state-machine + run command |
| run [goal] | Execute orchestrator toward a goal (generated by bootstrap) |

## Knowledge Base

```
knowledge/
  capabilities/ — 15 capability references (discovery, translate, tune, etc.)
  mappings/     — Tech stack mapping tables
  domains/      — Business domain knowledge
  learned/      — Cross-project experience (auto-populated)
```

## Entry Points

| Goal | Starting Point |
|------|---------------|
| New product (no code) | product-concept |
| Replicate/migrate code | discovery |
| Code governance | discovery + tune |
| Post-implementation | product-verify / visual-verify |
