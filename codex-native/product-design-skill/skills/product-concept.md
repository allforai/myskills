# Product Concept For Codex Native

Use this workflow to define the product problem, roles, value proposition, and
initial concept artifacts.

## Objectives

- clarify the problem space
- identify primary roles and value exchange
- define the concept baseline for downstream mapping

## Outputs

- `.allforai/product-concept/`

## Completion standard

- core problem and target roles are explicit
- concept artifacts are concrete enough to drive `product-map`
