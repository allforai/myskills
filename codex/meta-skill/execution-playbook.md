# Meta-Skill Execution Playbook

## Bootstrap

1. Read the project directory
2. Identify tech stacks, modules, architecture pattern
3. Ask user for goals (analyze / translate / tune / create / verify)
4. Select capabilities from `knowledge/capabilities/`
5. Generate node-specs + state-machine.json
6. Validate (structure + graph connectivity + safety)
7. Present summary for user confirmation
8. Write files

## Run (Generated)

The /run command is generated by bootstrap into the project.
It reads state-machine.json and orchestrates node execution.
See `knowledge/orchestrator-template.md` for the protocol.
