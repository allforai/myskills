# Game Design Discipline Subskills

## Goal

Add game-design subskills by planning discipline without wiring them into
bootstrap. These skills deepen the game design capability while keeping art,
runtime, and integration separate.

## Scope

- Add numeric/balance planning subskills.
- Add onboarding/FTUE planning subskills.
- Add content planning subskills.
- Add liveops/monetization planning subskills.
- Deepen narrative and level planning subskills.
- Do not modify bootstrap orchestration.
