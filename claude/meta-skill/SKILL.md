# Meta-Skill v0.6.2

> Unified skill generator — replaces 6 specialized plugins with /bootstrap + /run.

## Commands

| Command | Purpose |
|---------|---------|
| `/setup [check\|reset\|update]` | Detect and configure external capabilities (MCP tools + API keys) |
| `/bootstrap` | Analyze target project, generate node-specs + workflow + /run command |
| `/run [goal]` | Execute orchestrator toward a goal (generated by /bootstrap) |
| `/journal` | Record product decisions during development |
| `/journal-merge` | Merge decision batches |

## Architecture

```
Layer 1: Generator (/bootstrap)
  Lightweight analysis → knowledge loading → LLM free planning → node-specs + workflow.json

Layer 2: Orchestrator (/run)
  LLM-driven state machine → dispatch subagents → diagnosis on failure → safety guardrails
```
