# Use Case For Codex Native

Use this workflow to derive structured use cases from product and experience
artifacts.

## Objectives

- generate normal, exception, and edge cases
- provide both machine-readable and human-readable use cases

## Outputs

- `.allforai/use-case/use-case-tree.json`
- `.allforai/use-case/use-case-report.md`

## Completion standard

- core user goals have explicit use cases
- edge and failure paths are not omitted silently
