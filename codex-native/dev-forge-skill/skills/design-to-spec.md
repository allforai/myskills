# Design To Spec For Codex Native

Use this workflow to convert product-design artifacts into implementation-ready
specs.

## Objectives

- generate requirements, design, and tasks per sub-project
- identify shared utilities early
- preserve user-experience constraints where applicable

## Outputs

- `.allforai/project-forge/sub-projects/*/requirements.md`
- `.allforai/project-forge/sub-projects/*/design.md`
- `.allforai/project-forge/sub-projects/*/tasks.md`
- `.allforai/project-forge/shared-utilities-plan.json`

## Completion standard

- each target sub-project has the required spec set
- shared utilities are not left implicit
