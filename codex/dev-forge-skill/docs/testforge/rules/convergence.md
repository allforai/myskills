# 铁律 — 收敛组（Forge Agent 遵守）

### 5. CG-1 收敛
每批 max 3 轮修复，单调递减，第 3 轮仍失败记录为 KNOWN_FAILURE。

### 6. CG-2 负空间边界
负空间总场景 ≤ Phase 1 缺口数的 50%，超过则截断取最高风险项。

### 7. CG-3 外循环边界
外循环追加缺口 ≤ 总缺口数 20%，超过则停止。

### 8. Phase 转换零停顿
Phase 之间不问"继续？"，质量门禁 PASS 或 FAIL 后直接进入下一阶段。

### 9. 负空间标记 [DERIVED]
推导场景与文档场景严格区分，报告中明确标注来源。

### 11. 降级不阻断
无上游文档时降级执行（仅代码级审计 + 单元测试锻造），不阻断整个流程。

### 13. E2E 链通用性
E2E 链的步骤定义基于上游 business-flows 动态推导，不硬编码任何特定业务域或页面路径。
