# Product Verify For Codex Native

Use this workflow to validate implementation against the intended product
artifacts.

## Objectives

- compare implementation against upstream design intent
- combine static and dynamic acceptance findings
- produce follow-up tasks for mismatches

## Outputs

- `.allforai/product-verify/static-report.json`
- `.allforai/product-verify/dynamic-report.json`
- `.allforai/product-verify/verify-report.md`

## Completion standard

- acceptance findings are explicit
- unresolved mismatches are converted into actionable tasks
