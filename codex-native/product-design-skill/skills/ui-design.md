# UI Design For Codex Native

Use this workflow to generate or refine high-fidelity UI specifications after
structure is stable.

## Objectives

- define visual direction and tokens
- produce UI specs and previews
- keep fidelity aligned with upstream experience design

## Outputs

- `.allforai/ui-design/ui-design-spec.md`
- `.allforai/ui-design/preview/`

## Completion standard

- visual spec and previews exist
- major screens align with structure and interaction intent
