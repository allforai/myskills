# Code Replicate Core For Codex Native

Use this workflow to perform the core reverse-engineering pass over an existing
codebase.

## Objectives

- normalize input source and scope
- infer project type
- generate discovery and infrastructure profiles
- extract standardized `.allforai/` artifacts
- record fidelity limits and unresolved ambiguity

## Completion standard

- source and fidelity are explicit
- discovery artifacts exist
- extracted artifacts are written to the expected locations
- the final report states confidence limits and downstream next steps
