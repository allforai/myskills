# UI Forge — Codex Entry Point

This is the Layer 0 entry for the Codex-native ui-forge skill.

## What This Skill Does

`ui-forge` is a post-implementation UI refinement workflow for professional engineers.
It works on frontend screens that are already functionally complete, improving visual
quality or restoring design fidelity without changing business logic.

## When To Use

- The page works but looks rough
- The implementation drifted from design specs, tokens, or screenshots
- You want production-level visual quality without rewriting features

## When NOT To Use

- The page is not functionally complete yet (use dev-forge instead)
- You need to define product structure (use product-design instead)
- You need to implement new business features

## Key Principle

Restore fidelity first, then polish. Never polish on top of design drift.
