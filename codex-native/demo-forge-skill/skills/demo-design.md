# Demo Design For Codex Native

Use this workflow to turn upstream product artifacts into a realistic demo data
plan.

## Objectives

- determine entities and record volumes
- map app models and APIs
- surface API gaps that block realistic demo data
- define style assumptions relevant to media planning

## Outputs

- `.allforai/demo-forge/demo-plan.json`
- `.allforai/demo-forge/model-mapping.json`
- `.allforai/demo-forge/api-gaps.json`
- `.allforai/demo-forge/style-profile.json` when style inference is needed

## Completion standard

- planned entities exist
- record targets are non-zero
- major missing API gaps are called out explicitly
